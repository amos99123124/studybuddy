To solve the problem, we need to use the property of similar trapezoids. Similar trapezoids have corresponding sides that are proportional.

Given the trapezoids are similar, we can set up the following proportion based on the corresponding sides:

\[ \frac{45}{2x + 4} = \frac{40}{16} \]

First, simplify the right-hand side of the proportion:

\[ \frac{40}{16} = \frac{5}{2} \]

Now, we have the equation:

\[ \frac{45}{2x + 4} = \frac{5}{2} \]

To solve for \( x \), cross-multiply to get rid of the fractions:

\[ 45 \cdot 2 = 5 \cdot (2x + 4) \]

This simplifies to:

\[ 90 = 10x + 20 \]

Subtract 20 from both sides:

\[ 70 = 10x \]

Divide both sides by 10:

\[ x = 7 \]
