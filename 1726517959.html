We are given the equation \(0.9c + 1.89r = 17.01\) and need to show that the following equations are equivalent to it:

a. \(c = 18.9 - 2.1r\)

b. \(r = -\frac{10}{21}c + 9\)

Let's start with part (a):

We begin with the original equation:

\(0.9c + 1.89r = 17.01\)

To isolate \(c\), we first subtract \(1.89r\) from both sides:

\(0.9c = 17.01 - 1.89r\)

Next, we divide both sides by \(0.9\):

\(c = \frac{17.01 - 1.89r}{0.9}\)

Performing the division:

\(c = \frac{17.01}{0.9} - \frac{1.89r}{0.9}\)

\(c = 18.9 - 2.1r\)

Thus, we have shown that \(c = 18.9 - 2.1r\) is equivalent to the original equation.

Now, let's move to part (b):

We start again with the original equation:

\(0.9c + 1.89r = 17.01\)

To isolate \(r\), we first subtract \(0.9c\) from both sides:

\(1.89r = 17.01 - 0.9c\)

Next, we divide both sides by \(1.89\):

\(r = \frac{17.01 - 0.9c}{1.89}\)

Performing the division:

\(r = \frac{17.01}{1.89} - \frac{0.9c}{1.89}\)

\(r = 9 - \frac{0.9c}{1.89}\)

Since \(\frac{0.9}{1.89} = \frac{10}{21}\), we have:

\(r = 9 - \frac{10}{21}c\)

Thus, we have shown that \(r = -\frac{10}{21}c + 9\) is equivalent to the original equation.

In summary, we have demonstrated that both given equations are equivalent to the original equation \(0.9c + 1.89r = 17.01\).
