Let's determine if each set of ordered pairs on the graphs represents a function. A function is defined as a relation in which each input (x-value) has exactly one output (y-value). The most straightforward way to check this on a graph is by using the vertical line test.

The vertical line test states that if a vertical line intersects a graph in more than one point, the graph does not represent a function. Let's apply this test to each graph:
